# Laravel 5.3 and Angular 2 (release) example project.

## Installation:

```
composer install
npm install
```
Create `.env` file (can be based on `.env.example`
```
php artisan key:generate
```

## Building

```
npm run wpprod
```

## Watching

```
npm run wpdev
```

## Server

```
php artisan serve
```
